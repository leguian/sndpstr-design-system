SNDPSTR DS
==============

### Install SASS

NPM
```
npm install -g sass
```

Chocolatey (Windows alternative)
```
choco install sass
```

Homebrew (Mac OS X alternative)
```
brew install --devel sass/sass/sass
```
### Set up

Compile to your style sheet

```
cd style/
sass --watch sndpstr.scss:style.css
```
