SNDPSTR DS
==============

### Install SASS

NPM
```
npm install -g sass
```

Chocolatey (Windows alternative)
```
choco install sass
```

Homebrew (Mac OS X alternative)
```
brew install --devel sass/sass/sass
```
### Set up

Generate style.css

```
npm start
```
